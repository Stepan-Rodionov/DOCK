# Мой проект

Этот проект демонстрирует автоматическую проверку орфографии с использованием GitHub Actions.

## Описание

При каждом push в репозиторий, GitHub Action проверяет все Markdown файлы на правильность написания.

## Технологии

- GitHub Actions
- pyspelling
- Aspell
